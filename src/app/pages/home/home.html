Angular Rendering Demo

Демонстрационное приложение для сравнения CSR, SSR, SSG. По умолчанию уже включен Hydration.

Что здесь можно посмотреть

  • CSR — классический клиентский рендеринг. HTML приходит почти пустым, всё рисуется в браузере.
  • SSR — серверный рендеринг. HTML генерируется на сервере при каждом запросе.
  • SSG — статическая генерация. Страницы создаются во время build.
  • Hydration — “оживление” уже готового HTML без его повторного рендеринга.
